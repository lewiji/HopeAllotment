## Get started

After cloning the repository:

```sh
cd HopeAllotment/
```

Install dependencies:

```sh
npm install
```

Install the Gatsby CLI:

```sh
npm i --global gatsby-cli
```

Run the development server:

```sh
npm run start
```

Deploy (if you have permissions):

```sh
npm run deploy
```
